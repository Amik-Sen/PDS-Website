Dynamic Distribution Planning

An effective heuristic to solve large size inventory transportation problem of Indian PDS with supply disruptions due to COVID-19â€‹

Purpose of this application is to demonstrate a mathematical model for the public distribution system (PDS) of India considering three layered supply chain and disruptions due to COVID-19 in the PDS system, and solving the mathematical model using the heuristic approach developed. The three layered supply chain that we considered for this application is:

  • State Warehouses, which are basically godowns regulated by State Government
  • Wholesalers, Who get there goods from state warehouses and they send them to Fair Price Shops (FPS)
  • Fair Price Shops(FPS), are the stores which provide good to people at cheaper rates.

The current situation due to COVID-19 has affected the demands of fair price shops to address this change in demand, considering fair price shops, become an important part of the model.

Modules

These are back-bone of this application which provides us with all the components we require for generating desired results.

Forecast Module

The module responsible for producing expected demand, which is a vital component for our application.

Data collection Module

The module responsible for collecting input data or for updating existing data of the public distribution system.

Optimization Module

The module responsible for running the hybrid optimization algorithm to generate solution.

Visualisation Module

The module responsible for generating all the maps and graphs in this application.

What are the features we provide?

Based on our understanding of the public distribution system, we have created three features, which takes in consideration most of the factors that are involved in it's decision making. These easy to use features makes this application user friendly.
